Política de Privacidade

Este site não exige cadastro e não coleta dados pessoais diretamente.

1. Dados Coletados

Poderão ser coletadas informações técnicas de navegação, como:

  • Endereço IP
  • Tipo de navegador
  • Data e horário de acesso

Esses dados são utilizados exclusivamente para fins estatísticos e melhoria do site.

2. Plataformas Externas

Contribuições financeiras são realizadas por meio de plataformas externas, sujeitas às próprias políticas dessas plataformas.

Este site não armazena dados bancários ou financeiros.

3. Comunicação

Caso o usuário entre em contato por plataformas externas, como Discord, as informações fornecidas serão utilizadas apenas para fins de comunicação.

4. Compartilhamento

Não há venda ou compartilhamento de dados pessoais com terceiros.
